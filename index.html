Visao geral do problema

A pizzaria “Tá com fome” é especialista em desenvolver pizzas em formato de cone e pizzas tradicionais. As pizzas podem possuir os seguintes ingredientes: massa pizza tradicional, massa pizza cone, molho de tomate, queijo parmesão, queijo mussarela, queijo margherita, queijo provolone, queijo gongorzola, cebola e presunto. Atualmente o cozinheiro prepara as seguintes pizzas:

  • a. Mussarela: massa, molho de tomate, queijo mussarela, queijo gongorzola, cebola e presunto;
  • b. Quatro queijos: massa, molho de tomate, queijo parmesão, queijo mussarela, queijo margherita, queijo provolone, queijo gongorzola, cebola e presunto;

É importante ressaltar que a massa é modifica conforme o tipo da pizza, ou seja, caso a pizza seja tradicional é utilizado a massa pizza tradicional. Caso contrário, a massa escolhida é a massa pizza cone. Para melhorar o desempenho da pizzaria será desenvolvido um software para gerenciar o preparo das pizzas. O projetista especificou os seguintes etapas para tal funcionalidade:

  • Utilizar o padrão Fabrica Abstrata para instanciar os ingredientes;
  • Utilizar o padrão Builder será utilizado para criar as pizzas;
  • O padrão Builder deve utilizar o padrão fabrica abstrata;
  • O padrão Método Fabrica deve ser utilizado para implementa a escolha das pizzas;
  • O padrão Singleton para economizar a memória;
  • O padrão Protótipo para uma instanciação mais rápida; O código principal tem que apresentar uma codificação da instanciação de uma pizza de Mussarella e Quatro Queijos

Solução

Para a solução deste problema foi gerado o diagrama de classes demonstrado na figura acima. Foi utilizado todos os padrões propostos, e por exemplo, o padrão Singleton foi implemetado nos ingredientes, uma vez que não era necessario por exemplo, instanciar uma nova Cebola para colocar em duas pizzas que levam cebola, queijos iguais, molhos iguais. Padrão Fabrica foi utilizado para gerar todas as pizzas assim como a criação dos metodos necessarios. Deste modo, o codigo tornou-se um codigo menor, mais limpo e mais legivel.
